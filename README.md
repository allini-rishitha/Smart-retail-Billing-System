# Smart Retail Billing System

## Advanced Java Mini Project

### Technologies Used
- Java
- Servlets
- JDBC
- MySQL
- HTML
- JavaScript
- Apache Tomcat

### Project Description
Smart Retail Billing System is a web-based application developed using Advanced Java. The system allows users to add products, manage stock, add items to cart, generate bills, and automatically update inventory in the database after billing.

### Files Included
- Project Report
- Project Presentation
- DBConnection.java
- HomeServlet.java
- ProductServlet.java
- UpdateStockServlet.java
- index.html
